# ⚓ Корсары: Ветер свободы

> 🇬🇧 [Read in English](README.md)

Открытый ремейк **с нуля** в духе «Корсаров II» (Pirates of the Caribbean, Акелла, 2003) на движке **Godot 4.7**. Никаких оригинальных ресурсов игры не используется — весь код, данные и «графика» (процедурные корабли, шейдер воды) написаны заново.

## Что уже есть

- **Морской бой в 3D**: управление парусником с учётом ветра (левентик/бейдевинд/бакштаг/фордевинд), залпы бортов, 4 типа зарядов (ядра, книппели, картечь, бомбы) с разным профилем урона, перезарядка, ИИ противника, абордаж, потопление.
- **Глобальная карта** Карибского архипелага: 7 островов четырёх наций + пиратское поселение, переходы с расходом провизии и жалованием команде, случайные встречи.
- **Экономика**: 16 товаров, у каждой колонии свои экспорт/импорт, цены зависят от запасов, торговые маршруты прибыльны, навык торговли влияет на курс.
- **RPG-система**: 10 навыков (лидерство, фехтование, навигация, меткость, пушки, абордаж, защита, ремонт, торговля, удача), уровни, очки навыков.
- **Нации и репутация**: Англия, Франция, Испания, Голландия, пираты; войны, реакция на потопленные корабли, закрытые порты.
- **Квесты губернаторов**: доставка груза, охота на пиратов, пассажиры, сроки и штрафы.
- **Порт**: рынок, верфь (11 классов кораблей от тартаны до мановара, trade-in), найм команды, ремонт.
- **Сохранения** (JSON, `user://savegame.json`), автосейв после каждого перехода.

## Управление в бою

| Клавиша | Действие |
|---|---|
| W / S | Поднять / убрать паруса |
| A / D | Руль влево / вправо |
| Q / E | Залп левого / правого борта |
| R | Сменить тип заряда |
| B | Абордаж (вплотную к противнику) |

## Запуск

Нужен [Godot 4.x](https://godotengine.org/download) (бесплатный, ~100 МБ).

```bash
# macOS (Homebrew)
brew install --cask godot

# из папки проекта
godot --path .
```

Или откройте папку проекта в редакторе Godot и нажмите F5.

## Тесты

Вся игровая механика покрыта тестами (9 сьютов, ~960 проверок) и гоняется без окна:

```bash
godot --headless --path . -s tests/run_tests.gd     # юнит-тесты ядра
godot --headless --path . -s tests/smoke_scenes.gd  # smoke-обход всех сцен
```

Ядро (`core/`) не зависит от сцен: каждая система — чистый класс с детерминируемым RNG.

## Сборка под все платформы

Godot экспортирует один и тот же проект под **Windows 11/10, macOS, Linux и Web**. Пресеты уже настроены (`export_presets.cfg`).

1. Один раз скачайте шаблоны экспорта: в редакторе Godot — *Editor → Manage Export Templates → Download and Install*.
2. Экспорт из командной строки:

```bash
godot --headless --path . --export-release "Windows 11 (x86_64)" build/windows/corsairs.exe
godot --headless --path . --export-release "macOS (universal)"   build/macos/corsairs.zip
godot --headless --path . --export-release "Linux (x86_64)"      build/linux/corsairs.x86_64
godot --headless --path . --export-release "Web"                 build/web/index.html
```

`corsairs.exe` — обычное 64-битное приложение, работает на Windows 10/11 без зависимостей (движок и ресурсы упакованы в один файл). Web-сборку можно выложить на любой статический хостинг (itch.io, GitHub Pages).

## Структура проекта

```
core/      — игровая логика (без сцен, полностью тестируемая)
tests/     — тестовый фреймворк, юнит-тесты, smoke-тест сцен
scenes/    — main_menu, world_map, port, sea (3D-бой)
scripts/   — код сцен и автозагрузка Game
assets/    — шейдер воды
```

## Правовая заметка

Это фанатский ремейк-оммаж «с чистого листа». Проект не использует код, модели, текстуры, музыку или тексты оригинальной игры и не аффилирован с Акеллой. Если захотите запустить **оригинальные** «Корсары» на Windows 11 — купите игру (GOG/Steam) и посмотрите открытый движок [storm-engine](https://github.com/storm-devs/storm-engine), исходники которого официально открыты.
